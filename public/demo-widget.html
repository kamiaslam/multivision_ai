🎤 VoiceCake Widget Demo

🚀 Quick Start

Add the VoiceCake widget to your website with just one script tag:

<script src="https://your-domain.com/voicecake-widget.js" data-agent-id="53" data-position="bottom-right" data-theme="light"></script>
The widget is already loaded on this page! Look for the microphone button in the bottom-right corner.

⚙️ Configuration Options

Agent ID (Required)

The unique identifier for your AI agent:

data-agent-id="53"

Position Options

bottom-right
Default position
bottom-left
Left side of screen
top-right
Top right corner
top-left
Top left corner

Theme Options

light
Default theme
dark
Dark mode

📝 Complete Example

Here's a complete HTML page with the VoiceCake widget:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Website with Voice AI</title> </head> <body> <h1>Welcome to My Website</h1> <p>This page has a VoiceCake AI assistant!</p> <!-- VoiceCake Widget --> <script src="https://your-domain.com/voicecake-widget.js" data-agent-id="53" data-position="bottom-right" data-theme="light"></script> </body> </html>

🔧 Advanced Usage

Accessing the Widget Instance

You can access the widget instance for advanced control:

// Get widget status const status = window.voicecakeWidget.getStatus(); console.log('Widget status:', status); // Destroy the widget window.voicecakeWidget.destroy();

Multiple Widgets

You can have multiple widgets on the same page by creating them programmatically:

// Create additional widget instances const widget2 = new window.VoiceCakeWidget({ agentId: '97', position: 'bottom-left', theme: 'dark' });

📋 Requirements

  • Modern web browser with WebRTC support
  • HTTPS connection (required for microphone access)
  • Valid VoiceCake agent ID
  • Microphone permissions from user
Important: The widget requires HTTPS to access the microphone. Make sure your website is served over HTTPS in production.

🆘 Troubleshooting

Widget not appearing?

  • Check that the script URL is correct
  • Verify the agent ID is valid
  • Check browser console for errors

Microphone not working?

  • Ensure HTTPS is enabled
  • Check browser permissions
  • Try refreshing the page

Connection issues?

  • Check your internet connection
  • Verify the agent is active
  • Check browser console for WebSocket errors
